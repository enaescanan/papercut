Printing Duplex or Double-Sided

Using Microsoft Word


If Duplex Printing is Available
  1. Click the File tab.
  2. Click Print.
  3. Under Settings, click Print One Sided. If Print on Both Sides is available, your printer is set up for duplex printing.

If Duplex Printing is NOT Available

You can still print duplex, but you'll have to do it manually—printing the odd pages first and then the even pages next.

  1. Click the File tab.
  2. Click Print.
  3. Under Settings, click Print All Pages. Near the bottom, click Only Print Odd Pages.
  4. Click Print button.
  5. After the odd pages are printed, flip the stack over, and then under Settings, click Print All Pages. Near the bottom, click Only Print Even Pages.
  6. Click Print button.

Printing Multiple Pages Per Sheet

  1. Press [Ctrl]+P or [Cmd]+P to display the Print dialog box. Alternatively, choose Print from the File menu.
  2. From the Pages Per Sheet drop-down list, select the number of pages you want printed on each sheet of paper. Two pages per sheet is best if you are worried about small font size.
  3. Click OK to print the document.
